# Instagram-Clone
Insta Clone Flask Project

<pre>
Instagram
|--static
  |--stylesheets
  |--uploads
  |--images
|--templates
  |-feed.html
  |-profile.html
  |-userprofile.html
  ....
|-app.py
</pre>

## Setup

<h4>Step-1</h4>
<p>Create new Folder and create virtual environment</p>

```html
virtualenv .venv
```
<p>OR</p>

```python
python -m venv .venv
```
<h4>Step-2</h4>
<p>Activate Virtual Environment</p>

```bash
.venv\Scripts\activate
```

<h4>Step-3</h4>
<p>Install Modules/Packages</p>

```html
pip install flask flask-mysqldb
```
